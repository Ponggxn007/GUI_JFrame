# ระบบจัดการข้อมูลด้วย GUI JFrame

**GUI_JFrame** เป็นโปรเจกต์ที่พัฒนาด้วยภาษา Java โดยใช้ `JFrame` สำหรับสร้างส่วนติดต่อผู้ใช้งาน (Graphical User Interface) ซึ่งโปรเจกต์นี้ออกแบบมาเพื่อการจัดการข้อมูลแบบเรียลไทม์ โดยสามารถเพิ่ม, ลบ และแก้ไขข้อมูล พร้อมฟีเจอร์สำหรับการค้นหาข้อมูล รวมถึงแสดงผลลัพธ์ที่เข้าใจง่ายผ่านอินเทอร์เฟซที่เป็นมิตรกับผู้ใช้

---

## 🚀 ฟีเจอร์ของระบบ
- **เพิ่มข้อมูล**: รองรับการเพิ่มข้อมูลใหม่ลงในระบบ
- **แก้ไขข้อมูล**: แก้ไขข้อมูลที่มีอยู่ในฐานข้อมูลแบบเรียลไทม์
- **ลบข้อมูล**: ลบข้อมูลที่ไม่ต้องการได้อย่างง่ายดาย
- **การค้นข้อมูล**: สามารถค้นหาข้อมูลในระบบตามคำค้นหา
- **แสดงข้อมูล**: แสดงข้อมูลในรูปแบบตาราง เพื่อให้อ่านง่าย
- **ส่วนติดต่อผู้ใช้งานแบบกราฟิก**: ใช้งานง่าย รองรับการทำงานด้วยเมาส์และคีย์บอร์ด

---

## 💻 เทคโนโลยีที่ใช้
- **ภาษาโปรแกรม**: Java
- **เฟรมเวิร์ก GUI**: Swing (`JFrame` และส่วนประกอบอื่นๆ)
